Star rating example

Input options:

  • starsQuantity: number (default 5)
  • color: ThemePalette from Angular Material (default 'primary')
  • size: number (default 25)
  • value: number (if not use FormControl)
  • label: boolean (default false)
  • labelSize: number (default 20)
  • labelPosition: 'before' || 'after' (default 'before')
  • emptyIcon: string from Material Icon (default 'star_border')
  • halfIcon: string from Material Icon (default 'star_half')
  • fullIcon: string from Material Icon (default 'star')
  • static: boolean (default true)

Output options:

  • valueChange: number (EventEmitter if FormControl not use)
